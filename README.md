# 韵脚检查工具

一个用于分析和可视化中文歌词或诗歌中押韵模式的Web应用。

## 功能特点

- 🔍 **多类型押韵检测**：支持句尾押韵、句间押韵和句内押韵的自动检测
- 📊 **可视化报告**：生成清晰直观的押韵分析报告
- ⚙️ **可调节参数**：允许调整押韵检测的容差参数，以适应不同的创作风格
- 📱 **响应式设计**：适配不同屏幕尺寸，提供良好的移动端体验

## 技术栈

- **前端框架**：纯HTML、CSS、JavaScript
- **模块系统**：ES Modules
- **第三方库**：pinyin-pro（用于中文拼音转换）

## 使用方法

1. 在浏览器中打开 `index.html` 文件
2. 在左侧文本框中输入你的歌词或诗歌
3. 右侧将自动显示押韵分析结果
4. 可在下方调整检测选项，包括：
   - 检测句间押韵
   - 检测句内押韵
   - 调整各种押韵的容差参数

## 检测选项说明

- **检测句间押韵**：检测不同行之间的押韵关系
  - 间隔容差：允许押韵词语之间的间隔距离
  - 行差容差：允许押韵词语所在行的行号差异
- **检测句内押韵**：检测同一行内的押韵关系
  - 间隔容差：允许同一行内押韵词语之间的间隔距离
